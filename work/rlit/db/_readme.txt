файлы - tab-delimited, в кодировке Win1251

author - авторы
-id       уникальный id
-name     имя, фамилия (для соавторов - "имя, фамилия + имя, фамилия + ...")
-midname  отчество
-lang     язык (RU/EN)
-class    тип (А - автор, P - псевдоним, C - соавторы)
-nbook    кол-во книг (включая написанные в соавторстве)
-series   серия (см. ниже) или 0
начинающиеся с '#' - служебные: сборники, коллективы, неизвестные авторы


authorlink - связи автор/соавторы
-author   id соавторов/псевдонима
-linked   id связанного с ним автора 
для каждого псевдонима (P) - одна пара author-linked
для каждого соавтора (C) - столько пар author-linked, сколько авторов входит в соавторство
(!) когда псевдоним более известен, чем реальное имя - 
псевдонимом помечено реальное имя - например:
Акунин(A) - Чхартишвили(P)
Олди(A) - Ладыженский(P)
Олди(A) - Громов (P)


book - книги
- id      id книги
- author  id автора
- name    название книги
- lang    язык (RU/EN)
- series  серия или 0
- number  номер в серии или 0
- year    год написания или 0
- transl  переводчик 
- path    путь к файлу от корня библиотеки
- fname   имя файла (без .zip)
- size    размер файла в байтах
- asize   размер архива в байтах
- added   дата добавления/обновления книги (YYYMMDD)
- crc32   crc32 для файла (bigint)
(!) поле "переводчик" содержит только одного переводчика из fb2-тегов.
используется только для идентификации разных переводов.


series - серии
- id      id серии
- name    название серии
- class   тип A/G (может отсутствовать)
A - "активные" серии, которые раскладываются в отдельные папки
G - кросс-авторские серии и жанровые категории,
могут быть проставлены как у книг, так и у авторов.
по сути - путь к файлам от каталога "_" библиотеки


booktags - жанры
- id      id книги
- name    тег fb2 genre
все невалидные жанры приведены к схеме fb2.1, за исключением жанра other
у одной книги может быть несколько жанров


bookanno - аннотации
- id      id книги
- text    аннотация 
у одной книги может быть несколько аннотаций
перевод строки заменен на "\n"
длинные аннотации обрезаны до 4000 символов
конвертированные из unicode могут содержать мусор, встречаются недочищенные fb2-теги
